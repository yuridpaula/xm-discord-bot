# xm-discord-bot
Repositório destinado ao Projeto de Discord Bot do Grupo do Robô Xeque Mate.

# Prerequisites

* Must have installed Node JS
* Must have Git installed

# Setup

* To install dependencies: `npm install`

# Running Server

Simply run the command `node bot.js`
